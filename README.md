# weblayout-advanced-practice 🖥️

This repository contains practical assignments completed during the **Web Layout Advanced** course at **Skillbox**. It serves both as a portfolio for potential employers and as a reference for students learning modern frontend development techniques.

## Repository Structure

The code is organised by topics, each covering a specific advanced layout skill:

- **`grid/`** — Layout using CSS Grid.
- **`sass/`** — Styling with Sass (SCSS) preprocessor.
- **`pug/`** — Templating with Pug (formerly Jade).
- **`gulp/`** — Task automation with Gulp.js (includes `gulpfile.js`, `package.json`).
- **`letter weblayout sample_1/`** — Email layout example 1.
- **`letter weblayout sample_2/`** — Email layout example 2.

Each folder contains a complete, self-contained project demonstrating a specific tool or technique.
